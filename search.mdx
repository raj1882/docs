## API Endpoints

| Endpoint | Description | Try It |
|----------|-------------|--------|
| `/search?query=` | Keyword search by title | [Try](/search?query=admissions) |
| `/query` | Natural language Q&A | `POST http://localhost:8000/query` |
| `/toc` | Full table of contents | [View](/toc) |

## Example Queries

```bash
# Search by keyword
curl "http://localhost:8000/search?query=admissions"

# Ask natural language question
curl -X POST "http://localhost:8000/query" \
  -H "Content-Type: application/json" \
  -d '{"question": "What is the admissions policy?"}'
```
